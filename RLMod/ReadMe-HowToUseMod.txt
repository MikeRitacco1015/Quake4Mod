This mod increases all rocket launcher damage to 1500 (SP) and 1000 (MP)

1.  Copy folder to base Quake4 directory

2.  Add following line to launch options:

	+set fs_game RLMod +set developer 1

3.  Launch Game

4.  If you don't already have a rocket launcher, open console (ctrl + alt + ~), type following then enter, then close console:

	give weapon_rocketlauncher
